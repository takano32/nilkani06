## 技術メモ

### loadPastSessions() の設計方針

- `pastMessages`（過去、読み取り専用）と `allMessages`（現セッション、保存対象）を分離
- `saveSession()` は現セッション分のみ書き込むため、セッションファイルが肥大化しない
- Groq に渡す `channelHistory` は両者を結合してから `.slice(-MAX_HISTORY)` で切る

### RECENT.md の運用ルール

- セッション終了時に上書き（積み上げ不可）
- AGENTS.md の先頭で `@.agents/RECENT.md` としてインクルード
- Claude Code 専用（Codex の `@` 構文非対応は確認済み）
