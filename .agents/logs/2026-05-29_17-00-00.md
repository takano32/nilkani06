## 技術メモ

### Slack → Discord の対応関係

| Slack (旧) | Discord (新) |
|---|---|
| `@slack/bolt` の `App` | `discord.js` の `Client` |
| Socket Mode | Gateway (WebSocket) |
| `app.message(handler)` | `client.on('messageCreate', handler)` |
| `message.subtype \|\| message.bot_id` | `message.author.bot` |
| `message.channel` | `message.channelId` |
| `say(reply)` | `message.reply(reply)` |
| `SLACK_BOT_TOKEN` + `SLACK_APP_TOKEN` | `DISCORD_TOKEN` のみ |
| Slack API POST (form-encoded) | Discord Webhook POST (JSON) |

### discord.js の注意点

- `MessageContent` は Privileged Gateway Intent のため、Developer Portal で明示的に有効化が必要。
- DM に応答するには `Partials.Channel` を `partials` に追加する必要がある。
- `message.reply()` は Discord のリプライ UI（元メッセージへのリンク付き）になる。フラットに送りたい場合は `message.channel.send()` を使う。
- Discord のメッセージ上限は 2000 文字。長い返答は分割が必要（現状は未実装）。
