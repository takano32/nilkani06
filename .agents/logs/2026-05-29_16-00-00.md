## 技術メモ

### Groq への移行方針

`openai` npm パッケージはそのまま流用し、`baseURL` と API キーのみ差し替える。
Groq は OpenAI 互換 API を提供しているため `chat.completions.create` の呼び出し形式は変更不要。

```js
const openai = new OpenAI({
  apiKey: process.env.GROQ_API_KEY,
  baseURL: 'https://api.groq.com/openai/v1',
});
```

### boot.bash の bash 4.0+ 対応点

| 変更前 (POSIX sh) | 変更後 (bash 4.0+) |
|---|---|
| `#!/bin/sh` | `#!/usr/bin/env bash` |
| なし | `set -euo pipefail` |
| `[ -n "$A" ] && [ -n "$B" ]` | `[[ -n "$A" && -n "$B" ]]` |
| `text="$1"` (グローバル) | `local text="$1"` |
| `FIRST=1` / `[ "$FIRST" = "1" ]` | `first=true` / `if $first` |
| `node bot.js` | `node bot.js \|\| true` (`set -e` 対策) |

### サブスクリプションと API の違い

- ChatGPT Plus / Claude Pro はそれぞれ Web UI の利用料。API 利用は含まれない。
- Claude Code CLI は Claude Pro に含まれる（追加課金なし）。
- Slack Bot が実行時に呼ぶ API は別課金 → Groq 無料枠で代替。
