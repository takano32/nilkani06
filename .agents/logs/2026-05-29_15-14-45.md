## 技術メモ

### セッションファイルの形式 (`sessions/*.json`)

```json
{
  "session_start": "2026-05-29T06:14:45.000Z",
  "messages": [
    { "timestamp": "...", "channel": "C...", "role": "user",      "content": "こんにちは" },
    { "timestamp": "...", "channel": "C...", "role": "assistant", "content": "こんにちは！" }
  ]
}
```

- チャンネルごとに履歴をフィルタして OpenAI に渡す
- `role` は OpenAI の `user` / `assistant` をそのまま使用
- `channel` フィールドで複数チャンネルを1ファイルに同居させている

### Slack 通知 (boot.sh)

```sh
curl -s -X POST "https://slack.com/api/chat.postMessage" \
  -H "Authorization: Bearer ${SLACK_BOT_TOKEN}" \
  --data-urlencode "channel=${SLACK_NOTIFY_CHANNEL}" \
  --data-urlencode "text=${text}"
```

form-encoded で送るため JSON 組み立て不要。POSIX sh 互換。
